# Videojuego-Pygame

## Descripción del juego
**Rocket Ascent: Mega Fusion Gold Edition** es un juego arcade hecho con **Pygame** en el que controlas un cohete que asciende constantemente mientras esquivas peligros y recoges monedas.

Durante la partida:
- El cohete se mueve con inercia (aceleración, fricción y gravedad ligera).
- Debes evitar obstáculos (meteoritos) que aparecen en el recorrido.
- Puedes recolectar monedas para aumentar tu puntuación.
- Al juntar suficientes monedas, se activan minijuegos aleatorios con distintos retos.

El objetivo es sobrevivir el mayor tiempo posible, subir de altura y conseguir la mayor cantidad de monedas.

## Controles básicos
- **Mover izquierda/derecha:** `A / D` o flechas `← / →`
- **Mover arriba/abajo:** `W / S` o flechas `↑ / ↓`
- **Iniciar partida:** `Espacio`

## Ejecución
1. Asegúrate de tener Python y Pygame instalados.
2. Ejecuta:

```bash
python main.py
```
